# Margin Notes

Enclose margin notes with an opening `:::aside` and a closing `:::`.

## Simple Margin Note

```
:::aside

Some Margin Note

:::
```

## Extended Margin Notes

If margin note overlaps content, adjust height of margin note by adding a number from 1 to 5 like this

```
:::aside:2

Some Margin Note

Some Margin Note

Some Margin Note

:::
```
