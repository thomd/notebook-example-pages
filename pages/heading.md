# Heading

Each headline section is **editable** individually using a `[ Edit ]` link right next to it. 

Editing respects the heading hierarchy.

```
# h1 Headline

## h2 Headline

### h3 Headline

#### h4 Headline
```

# h1 Headline

text

## h2 Headline

text

### h3 Headline

text

#### h4 Headline

text
