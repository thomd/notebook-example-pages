# Lists

## Unordered Lists

```
- item 1
- item 2
  - item 2.1
  - item 2.2
- item 3
```

- item 1
- item 2
  - item 2.1
  - item 2.2
- item 3

## Ordered Lists

```
1. item 1
1. item 2
   - item 2.1
   - item 2.2
1. item 3
```

1. item 1
1. item 2
   - item 2.1
   - item 2.2
1. item 3

## Lists with Code

1.  first

        git add .

1.  second

        git commit -m "a meaningful message"

1.  third

        git push
