# Blockquote

## Single Line Blockquote

```
> blockquote
```

> blockquote

## Multi Line Blockquote

Add **two spaces** at the end of the blockquote line (except the last line)

```
> blockquote[space][space]
> blockquote[space][space]
> blockquote
```

> blockquote  
> blockquote  
> blockquote
